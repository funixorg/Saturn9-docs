# `strcpy`

Copy a string into another string
## Syntax

```C
char* strcpy(
	[out] char*        dest,
	[in] const char*   src
);
```

## Parameters

`[out] char* dest`

Where stored the copy of *string*

`[in] const char* src`

Source of *string* under *pointer*

## Return value

Returns the parameter `char* dest`
