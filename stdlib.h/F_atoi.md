# `atoi`

Convert string to integer
## Syntax

```C
int atoi(
	[in] const char  *str,
	[in] unsigned    base,
  );
```

## Parameters

`[in] const char *str`

*String* under *pointer* to be converted

`[in] unsigned base`

The base from which conversion starts

## Return value

Returns the conversion made with the base put in parameter
