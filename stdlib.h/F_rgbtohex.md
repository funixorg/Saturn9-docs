# `rgbtohex`

## Syntax

```C
uint32_t rgbtohex (
	[in] unsigned int   R,
	[in] unsigned int   G,
	[in] unsigned int   B,
);
```

## Parameters

`[in] unsigned int R`

*Red* channel

`[in] unsigned int G`

*Green* channel

`[in] unsigned int B`

*Blue* channel

## Return value

Returns the color in hexadecimal with the prefix `0x`, for example: `0xffff`
